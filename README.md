## 🔮 Manifesto

  

**Flourishance** is not a to-do list. It is a **Personal Governance System** designed to bridge the gap between who you are and who you intend to be.

  

Built on principles of **Identity-Based Habits**, **Stoicism**, and **System Dynamics**, Flourishance replaces rigid planning with dynamic protocols. It leverages Large Language Models (LLMs) not just to generate text, but to act as a **Chief of Staff**—analyzing your decisions, simulating future trajectories, and auditing your weekly performance.

  

>  *"We do not rise to the level of our goals. We fall to the level of our systems."*

  

---

  

## ⚡ Core Architecture

  

Flourishance is divided into five distinct engines that work in unison to regulate behavior and strategy.

  

| Module | Engine Name | Function & Mechanics |
| :--- | :--- | :--- |
| 🧬 |  **Identity Kernel**  |  **The Foundation.** Instead of goals, you build Identities.<br>• **Proof-Based Voting:** Every action is a vote for your future self.<br>• **Decay Logic:** Strength score drops if neglected (48h half-life). |
| ⚡ |  **Action Protocol**  |  **The Execution Layer.** Dynamic task stack.<br>• **Context Aware:** Adapts to Morning/Evening & High/Low energy.<br>• **Auto-Balancing:** Injects tasks for neglected life areas (Health, Wealth, etc). |
| 🧭 |  **Compass**  |  **The Strategy Suite.** AI-powered cognitive tools.<br>• **Trajectory Simulator:** Predicts 1-year & 10-year habit consequences.<br>• **Pathfinder:** Generates high-ROI 80/20 roadmaps.<br>• **Decision Core:** Resolves dilemmas against your Constitution. |
| 🛡️ |  **Defense Grid**  |  **The Immunological System.**<br>• **Interrupter Pattern:** Physiological resets (Box Breathing) for bad loops.<br>• **Risk Radar:** Visualizes threat levels based on recent slips. |
| ✨ |  **Neural Spark**  |  **The Wisdom Engine.**<br>• **Combinatorial Insight:** Merges domains like Game Theory & Biology.<br>• **Synthesizer:** Generates novel mental models for complex problems. |

## 🧠 AI Integration

  

Flourishance is designed to be **Model Agnostic**. It does not rely on hard-coded heuristics but instead streams context to an LLM to generate personalized governance advice.

  

You can power Flourishance with **any state-of-the-art model** (e.g., GPT-4o, Claude 3.5, DeepSeek, Gemini, Llama 3) via an OpenRouter connection.

  

### The Persona System

  

| Context | Assigned Persona | objective |
| :--- | :--- | :--- |
|  **Strategy**  |  `Chief of Staff`  | Binary decision making, rigorous analysis, 80/20 thinking. |
|  **Simulation**  |  `Prediction Engine`  | Extrapolates second and third-order consequences of actions. |
|  **Review**  |  `Performance Coach`  | Brutally honest feedback on governance leaks and drift. |
|  **Insight**  |  `Curator of Wisdom`  | Connects obscure concepts (Physics, History) to daily life. |

## 🚀 Getting Started

  

### Prerequisites

- Node.js (v18+)

- An API Key from [OpenRouter](https://openrouter.ai/) (Recommended for model flexibility) or a direct provider key.

  

### Installation

  

1. **Clone the repository**

```bash

git clone https://github.com/yourusername/flourishance.git

cd flourishance

```

  

2. **Install dependencies**

```bash

npm install

```

  

3. **Configure Environment**

Create a `.env` file in the root directory. You can choose any provider, but the app is optimized for routing via OpenRouter to allow switching models easily.

```env

# The gateway to your preferred Intelligence Model

VITE_OPENROUTER_API_KEY=sk-or-your-key-here

```

  

4. **Run Local Server**

```bash

npm run dev

```
